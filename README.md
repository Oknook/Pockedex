# Pockedex
Pockedex for PockeRogue challenges

### 1일차
확장프로그램 데모프로젝트 생성 -> 개발자 모드로 크롬에서 실행
popup파일 수정으로 다크모드, 슬라이드 버튼 추가
기본 버튼(P)과 수정기능 추가
설정된 버튼 클릭 시 사각형 도형 출력 확인
사각형 도형에 재입력 or X클릭 시 끄는 기능 추가
